ranger kms
==========
it is used to store keys in database.

main funtions are:
1)Key management - provides the ability to create, update or delete keys using Ambari or rest API.
2)Access Control Policies - provides the ability to manage access control policies within ranger kms.
 Access control policies generate or manage keys, adding another layer of security for data encrypted in hadoop.
3)Audit - provides a full audit trace of all actions that are done by ranger kms


list delete - others possible

read, copy - not possible

when ranger is used to encrypt a directory, users without ownership can list and delete the contents.
read and copy is not possible for users without ownership
